?? PASOS PARA EMPEZAR (Solo 3 pasos)
??????????????????????????????????????????????????????????????

PASO 1: INSTALAR MYSQL
??????????????????????
1. Ve a: https://dev.mysql.com/downloads/mysql/
2. Descarga MySQL Community Server
3. Instala con configuración por defecto
4. ?? IMPORTANTE: Recuerda la contraseña de root que configures


PASO 2: CREAR LA BASE DE DATOS
??????????????????????????????
1. Abre MySQL Workbench
2. Conecta a tu servidor local
3. Abre el archivo: database_script.sql
4. Presiona: Ctrl + Shift + Enter (ejecutar todo)
5. ? Verás mensaje: "Query OK"


PASO 3: CONFIGURAR CONTRASEÑA
?????????????????????????????
1. Abre el archivo: DatabaseConnection.cs
2. Busca la línea 9
3. Cambia esta parte:
   
   Pwd=;
   
   Por tu contraseña:
   
   Pwd=tu_contraseña_mysql;

4. Guarda el archivo (Ctrl + S)


?? ¡LISTO! AHORA EJECUTA LA APLICACIÓN

? PROBLEMAS COMUNES Y SOLUCIONES
??????????????????????????????????????????????????????????????

? "No se puede conectar a la base de datos"
? SOLUCIÓN:
   1. Abre Servicios de Windows (Win + R ? services.msc)
   2. Busca "MySQL80"
   3. Click derecho ? Iniciar
   
   O desde CMD como administrador:
   net start MySQL80


? "Authentication failed"
? SOLUCIÓN:
   Verifica la contraseña en DatabaseConnection.cs línea 9
   Debe coincidir con tu contraseña de MySQL


? "Unknown database 'concesionaria'"
? SOLUCIÓN:
   Ejecuta el script database_script.sql en MySQL Workbench

?? COMANDOS ÚTILES
??????????????????????????????????????????????????????????????

Iniciar MySQL (CMD como Administrador):
> net start MySQL80

Detener MySQL:
> net stop MySQL80

Hacer backup de la base de datos:
> mysqldump -u root -p concesionaria > backup.sql

Restaurar base de datos:
> mysql -u root -p concesionaria < backup.sql

?? VERIFICACIÓN RÁPIDA EN MYSQL WORKBENCH
??????????????????????????????????????????????????????????????

1. Abre MySQL Workbench
2. Conecta a tu servidor local
3. Ejecuta estas consultas:

   -- Ver bases de datos
   SHOW DATABASES;
   
   -- Debe aparecer 'concesionaria'

   -- Seleccionar la base de datos
   USE concesionaria;
   
   -- Ver tablas
   SHOW TABLES;
   
   -- Debe aparecer 'coches'

   -- Ver todos los vehículos
   SELECT * FROM coches;
   
   -- Debe mostrar 5 vehículos
